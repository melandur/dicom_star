# Dicom*

[![GitHub License](https://img.shields.io/github/license/melandur/dicom_star)](LICENSE)

## Description

Dicom* is a customizable sequence filter based on dicom tags.

## Installation

```bash
git clone https://github.com/melandur/dicom_star.git
```
```bash
cd dicom_star
```
```bash
poetry install
```
```bash
python main.py
```
